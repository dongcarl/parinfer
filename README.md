# Parinfer for Emacs

A [Parinfer] package for [Emacs].
> Unfinished, WIP.

## Description

This is a file-by-file port of [Parinfer] from Clojure to Emacs Lisp, the end goal is to have a fully working implementation for the [Emacs] text editor.
